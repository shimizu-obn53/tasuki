# TASUKI - 美容院・整骨院 事業承継マッチング

## ローカルで動かす手順

### 1. Node.jsのインストール確認
```bash
node -v   # v18以上推奨
npm -v
```

### 2. パッケージインストール
```bash
cd tasuki
npm install
```

### 3. 開発サーバー起動
```bash
npm run dev
```

ブラウザで http://localhost:3000 を開く

---

## ページ構成

| URL | 内容 |
|-----|------|
| `/` | トップページ（ランディング） |
| `/listings` | 案件一覧（フィルター付き） |
| `/listings/[id]` | 案件詳細 |
| `/register` | 売り手登録フォーム |
| `/about` | TASUKIとは |
